# Supabase Todo Demo

## Purpose
Minimal hello-world project for Supabase as a Backend as a Service.

## Features
- add a todo
- load todos from the database
- display todos in the frontend

## Tech Stack
- React + Vite
- Supabase
- PostgreSQL

## Setup
1. Create a Supabase project
2. Create the `todos` table
3. Enable RLS
4. Add read/insert policies
5. Add `.env` values
6. Run:
   npm install
   npm run dev

## Notes
This is a minimal demo. The policies are intentionally simple and not meant for production use.
